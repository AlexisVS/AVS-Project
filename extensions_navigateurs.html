Mes extensions de navigateurs

Voici une liste d'extensions que j'utilise pour le développement WEB et pour mon quotidien




Extensions pour le Developpement

  • Clear Cache

    Une extension qui permet en un clique de vider les caches de son navigateur


  • Font finder

    Permet d'avoir tous les attributs CSS d'une police.
    Cliquez sur l'extension et puis cliqué sur le texte de votre choix.


  • JetBrains IDE Support

    Une extension développé par la société JetBrains qui développe des IDE spécifiques à chaque type de développemnt


  • Viewport Dimensions

    Extensions qui permet de savoir la résolution de votre viewport.
    Très utile pour faire du responsive


  • Material Design Color Palette

    Est une extension qui permet d'avoir une palette de couleur prete à être copier pour être inserer dans nos feuille de style

    C'est une extensions qui permet de vous donné la définition d'un mot que vous aurez sélectionné au préalable


  • Dark Background and Light Text

    Une extension incroyable qui permet de changer automatiquement la couleurs des sites à votre guise


  • uBlock Origin

    un bloqueur de pub


  • Mate Translate

    Un traducteur rapide.
